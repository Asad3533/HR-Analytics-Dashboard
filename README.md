# 📊 HR Analytics Dashboard in Microsoft Excel

## 📌 Project Overview

This project is an interactive **HR Analytics Dashboard** built using **Microsoft Excel 2019**. The dashboard analyzes employee data to provide insights into workforce distribution, employee attrition, salary trends, and other key HR metrics.

The goal of this project is to demonstrate advanced Excel skills commonly required for **Data Analyst** roles.

---

## 🎯 Objectives

* Analyze employee attrition and workforce distribution.
* Identify salary trends across departments and job roles.
* Monitor employee demographics and work patterns.
* Build an interactive dashboard using Pivot Tables, Pivot Charts, and Slicers.

---

## 🛠 Tools & Technologies

* Microsoft Excel 2019
* Power Query
* Pivot Tables
* Pivot Charts
* Slicers
* Excel Tables
* XLOOKUP
* INDEX + MATCH
* Dynamic Array Functions (UNIQUE, FILTER, SORT)

## 📂 Dataset

**Dataset:** IBM HR Employee Attrition Dataset

The dataset contains employee-related information including:

* Department
* Job Role
* Education Field
* Monthly Income
* Attrition Status
* Overtime
* Gender
* Marital Status
* Years at Company
* Total Working Years
* Job Satisfaction
* Performance Rating
